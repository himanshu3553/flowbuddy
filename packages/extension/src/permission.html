If it keeps failing on macOS

  1. Open System Settings → Privacy & Security → Microphone.
  2. Enable Google Chrome.
  3. Fully quit and reopen Chrome, then click Retry.
  4. Still blocked? Visit chrome://settings/content/microphone and remove this extension from the “Not allowed” list.
