A compact mono metadata chip for captured-layers, step counts, and routes.

```jsx
<Tag>Screen</Tag> <Tag>Voice</Tag> <Tag>DOM</Tag>
<Tag tone="brand">/settings/security</Tag>
<Tag>6 steps</Tag>
```

- **tone**: `neutral` (gray) or `brand` (indigo-tint, for routes / brand metadata).
- Always mono. Group captured-layer tags in a flex row with `gap: 8px`.
